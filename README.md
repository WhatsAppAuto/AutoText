# Auto-Text
An Automated Broadcasting Agent for Whatsapp-Web

### Requirements
    Python >= 3.x
    Flask >= 1.1.2
    Selenium >= 3.141.0
    Pandas >= 1.0.0
    
## CSV Format
    ```bash
    <name><number>
    ```

## Installation

1. Clone and navigate to Auto-Text directory.

2. Run the Flask Server.<br>
   
    ```bash
    python3 app.py
    ```

3. You're all done :)
4. To quit ctrl+c.
